# Hexflower Weather for Forbidden Lands

Criei esta página com o intuito de me ajudar a controlar a temperatura nos meus jogos de Forbidden Lands. As regras aqui apresentadas são fruto de homebrew.

## Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias web:

*   **HTML5**: Para a estrutura da página.
*   **CSS3**: Para a estilização, utilizando Flexbox e CSS Variables para um design moderno.
*   **JavaScript (ES6+)**: Para toda a lógica da aplicação, incluindo a geração da grade hexagonal, interatividade e manipulação do DOM. O projeto utiliza JavaScript puro (Vanilla JS), sem a necessidade de frameworks externos.
*   **SVG (Scalable Vector Graphics)**: Para a renderização da grade hexagonal, permitindo que os hexágonos sejam escaláveis e interativos.

## Como Utilizar

Para utilizar a ferramenta, basta abrir o arquivo `index.html` em qualquer navegador de internet moderno.

## Estrutura do Projeto

*   `index.html`: O arquivo principal da página.
*   `style.css`: Contém todos os estilos da aplicação.
*   `script.js`: Onde se encontra a lógica principal para a geração e interatividade da grade.
*   `hexagon_data.js`: Arquivo com os dados de cada hexágono, como imagens e textos.
*   `img/`: Pasta com todas as imagens utilizadas nos hexágonos.
*   `README.md`: Este arquivo.
